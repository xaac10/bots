# Discord Aternos Bot

Este bot de Discord te permite encender tu servidor de Aternos con el comando `!encender`.

## Instalación

1. Clona el repositorio
2. Instala las dependencias con `npm install`
3. Crea un archivo `.env` usando `.env.example` como guía
4. Ejecuta el bot con `npm start`
5. Para producción en Koyeb, configura las variables de entorno desde el panel de Koyeb.

## Variables de entorno

Debes definir estas variables en tu archivo `.env` o en el panel de Koyeb:

```
DISCORD_TOKEN=tu_token
ATERNO_USER=tu_usuario_aternos
ATERNO_PASS=tu_contraseña_aternos
```
